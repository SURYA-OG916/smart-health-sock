# Smart Health Monitoring Sock

An IoT-based smart sock that monitors health parameters using sensors.

## Features
- Heart rate monitoring (MAX30102)
- Sweat detection using humidity sensor
- Foot temperature monitoring
- Data sent to smartphone app

## Hardware Used
- MAX30102 Pulse Sensor
- Humidity Sensor
- Arduino Nano / ESP32
- Bluetooth Module
- Conductive thread

## Applications
- Athlete monitoring
- Diabetic foot monitoring
- Elderly health monitoring
